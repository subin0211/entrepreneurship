인수 & 합병 창업 = M & A

    인수란?
    - 한 기업이 다른기업 취득하여 경영권 획득하는 것
      ※ A + B = A

    합병이란?
    - 두 개 이상의 기업이 하나의 기업으로 합쳐지는 것
      ※ A + B = A or B
      ※ A + B = A + B
      ※ A + B = C

  • 장점 : 기존의 장점 유지
  • 단점 : 기존의 단점 + 추가 비용
